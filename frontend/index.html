Patch notes

  • v0.1 — Patch 16.9 — rank brackets May 7, 2026 Pick rates now ship for six rank brackets (Silver, Gold, Platinum, Emerald, Diamond, Master+) on patch 16.9. The "Pick-rate patch" control in the sidebar is now a Rank bracket selector — switching it re-weights matchup/synergy scoring with that bracket's playrates and reloads the strength-curve reference distribution to match. Default is Diamond.

    Heads up: historical patches (16.1–16.8) are no longer selectable. The app now ships only the current patch's data across the six rank brackets — past patches have been dropped.
