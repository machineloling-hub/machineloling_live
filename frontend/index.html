Patch notes

  • v0.1 — Patch 16.9 — rank brackets May 7, 2026 Pick rates now ship for six rank brackets (Silver, Gold, Platinum, Emerald, Diamond, Master+) on patch 16.9. The "Pick-rate patch" control in the sidebar is now a Rank bracket selector — switching it re-weights matchup/synergy scoring with that bracket's playrates and reloads the strength-curve reference distribution to match. Default is Diamond.

    Heads up: historical patches (16.1–16.8) are no longer selectable. The app now ships only the current patch's data across the six rank brackets — past patches have been dropped.

Individual Champ Compare

What each column means (click to collapse)
Total Overall, how similar is this champion to the one you picked? +1 = nearly identical playstyle profile, 0 = unrelated, −1 = total opposite. Matchup Do these two champs struggle into the same enemies and beat up the same enemies? High = same kinds of opponents counter them and the same kinds of opponents feed them. Negative = one's strong matchup is the other's bad matchup. Synergy Do they like the same kinds of teammates? High = same partners boost both. Negative = the things that make one champ shine make the other worse. Blind Δz Is this champ safer to blind-pick than the one you picked? Positive = more reliable across the board (fewer hard counters). Negative = more polarized — bigger swings, more matchup-dependent. Both Strong The top 3 matchups/teammates where both champs have a winrate edge. The cells where they both shine — pick either one into these. Both Weak The top 3 matchups/teammates where both champs lose winrate — their shared blind spots. If both your options struggle here, you need a third pick. Most Different The top 3 places these two champs disagree — one wins where the other loses. Useful for figuring out when to swap between them: if you're up against one of these matchups, the other champ is probably your better answer.

Column definitions — detailed (click to collapse)
Total The average of the Matchup and Synergy correlation columns. Matchup Pearson correlation of the two champions' winrate-edge (Δ pp) vectors across every eligible opponent in all 5 roles. PR-weighted by opponent pick rate if the sidebar checkbox is on. Synergy Same correlation as Matchup, but over Δ pp with every eligible partner across the 4 other roles. Blind Δz The row champion's aggregate blindability z minus the selected champion's. Same z-scale as the Blindability tab; high = consistent across opponents and partners. Both Strong Cells where both Δ pp values are positive, sorted by min(Δ_selected, Δ_row) descending — biggest mutual upside first. Both Weak Cells where both Δ pp values are negative, sorted by max(Δ_selected, Δ_row) ascending — the cells where both champs are most in the red. Most Different Cells where the signs of the two Δ pp values are opposite, sorted by |Δ_selected − Δ_row| descending.
A "cell" in the last three columns is one matchup or synergy slot — e.g. vs SUP_Yuumi or w/ ADC_Lucian. Toggle Show individual deltas above the table to also see the raw winrate-edge numbers (the selected champ's, then the row champ's) inside each cell.
Driven by the live API — change PR floor, patch, or PR-weighted in the sidebar and this updates. Correlation is over each champion's delta-pp vector across every eligible matchup (or partner). Blindability is the same aggregate z used by the Blindability tab.
